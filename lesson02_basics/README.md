# 第二堂：資料型態與變數

重點涵蓋變數、命名規則、基本資料型態、字串操作、數學與邏輯運算、運算優先順序，以及 Python 的縮排語法。

## 學習目標
- 認識賦值：左邊是變數、右邊是值，舊值會被覆蓋
- 能判斷合法的變數命名、知道保留字與大小寫敏感
- 了解四大基本型態 (int/float/str/bool) 與型態轉換
- 熟悉字串串接、f-string，並避免型態錯誤
- 操作常見運算子：+ - * / // % **，比較運算，指定運算子 (+=)
- 掌握運算優先順序：括號 > 次方 > 乘除 > 加減
- 理解縮排錯誤 (IndentationError) 的成因

## 檔案說明
- `exercises.py`：小試身手題目，請依提示完成並執行
- `solutions.py`：參考解答，含簡短說明與預期輸出

## 建議使用方式
1. 先閱讀本頁目標與對應投影片概念
2. 開啟 `exercises.py`，逐題撰寫並執行確認結果
3. 完成後對照 `solutions.py`，比對邏輯與輸出
4. 嘗試修改數值或字串，觀察型態變化與錯誤訊息
